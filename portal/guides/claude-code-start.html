מה זה Claude Code?

תוכנה חכמה מאוד שמבינה עברית ויודעת לכתוב בשבילך קוד אמיתי — אתרים, משחקים, אפליקציות.

במקום ללמוד שפות תכנות מסובכות, אתה פשוט מתאר מה אתה רוצה, ו-Claude בונה את זה. כמו מורה פרטי למתכנתים.

💡 דוגמה: "תבנה לי דף עם כותרת כחולה וכפתור שאומר שלום" — וזה קורה!
📦

מה צריך להתקין?

4 דברים קטנים. כל אחד לוקח 2-5 דקות:

1

Node.js

המנוע שמפעיל את Claude Code

2

Git

כלי שומר גרסאות — חוזר אחורה כשטועים

3

Claude Code

הכלי עצמו — פקודה אחת בטרמינל

4

חשבון Anthropic

התחברות עם Google, חינם להתחלה

🟢

1. התקנת Node.js

היכנס לאתר הרשמי ולחץ על הכפתור הירוק:

nodejs.org

🍎 Mac

הורד את ה-.pkg, לחץ פעמיים, Next → Next → Install

🪟 Windows

הורד את ה-.msi, לחץ פעמיים, אשר את כל השלבים

💡 בקש עזרה מהורה! זה מותקן על המחשב רק פעם אחת.
💾

2. התקנת Git

Git זה כמו "שחזור" — אפשר לחזור אחורה בכל רגע אם משהו נשבר.

git-scm.com/downloads

🍎 Mac

פתח את Terminal (חיפוש → "Terminal") והקלד:

git --version

אם יש מספר — מותקן! אם לא, זה יציע להתקין.

🪟 Windows

הורד את ה-.exe, לחץ Next על הכל (ברירות מחדל בסדר).

✨

3. התקנת Claude Code

פתח את Terminal (מק) או PowerShell (ווינדוס) והרץ:

npm install -g @anthropic-ai/claude-code

ההתקנה לוקחת 1-2 דקות. תראה הרבה שורות — זה נורמלי!

💡 המפתח: הקלד ← npm install. אל תקליד $. הסימן $ רק מראה שזו פקודה בטרמינל.
🔑

4. התחברות לחשבון

הרץ את הפקודה:

claude

ייפתח דפדפן — לחץ Continue with Google. אם אין לך חשבון Anthropic, הוא ייווצר אוטומטית.

💡 השלב הזה הכי חשוב! בלעדיו Claude לא יעבוד. בקש עזרה מההורים אם צריך.
⚠️ חשוב: אל תשתף עם אף אחד את הסיסמה או את הקוד שמופיע לך. זה אישי לך בלבד!
🎉

השיעור הראשון!

צור תיקייה חדשה ותיכנס אליה:

mkdir my-first-project cd my-first-project git init claude

עכשיו אתה בתוך Claude Code. תקליד:

תבנה לי דף אינטרנט עם כותרת "שלום עולם" וכפתור ירוק

Claude יבנה בשבילך את האתר! תוך 30 שניות יהיה לך דף אמיתי.

🥇

4 כללי זהב לעבודה נכונה

1

תמיד Git קודם

לפני שמבקשים מ-Claude לשנות משהו — תעשה git commit. ככה תוכל לחזור אחורה אם יתקלקל משהו.

2

פרומפט ברור וספציפי

במקום "תתקן" → "תוסיף כפתור ירוק בפינה הימנית שיגיד שלום". ככל שתהיה ברור יותר, התוצאה תהיה מדויקת יותר.

כללי זהב — המשך

3

אל תאשר עיניים עצומות

Claude יראה לך מה הוא רוצה לשנות לפני שהוא משנה. תקרא את זה! אם לא מבין — שאל אותו להסביר. אתה הבוס.

4

שומרים אחרי כל הצלחה

עבד לך משהו? git add . ואז git commit -m "תיאור". ככה לא מאבדים את ההצלחה הבאה.

🛠️

3 פקודות חובה להכיר

1

claude

מפעיל את Claude Code. מכאן מתחילים לעבוד.

2

git status

מראה לך מה השתנה בפרויקט שלך.

3

git commit -m "מה עשיתי"

שומר את השינויים בצורה קבועה. כתוב תיאור קצר של מה שינית.
